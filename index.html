What does the HTML <a> tag define and how is it commonly used in web development?

The HTML <a> tag defines a hyperlink. It is used to link from one page to another. The href attribute specifies the destination address of the link.

What is the difference between a class selector and an ID selector in CSS, and when should each be used?

Class selectors use a period (.) and can be applied to multiple elements, making them ideal for reusable styles. ID selectors use a hash (#) and must be unique on the page, making them suitable for specific one-off elements. Class selectors have lower specificity than ID selectors.

css flexbox layout

In CSS Flexbox, what is the purpose of the justify-content property and what are some of its possible values?

The justify-content property aligns flex items horizontally within the container. Possible values include: flex-start (items align to the left), flex-end (items align to the right), center (items align at the center), space-between (items display with equal spacing between them), space-around (items display with equal spacing around them), and space-evenly (items display with equal spacing between and around them).
